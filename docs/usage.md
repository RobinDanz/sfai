# Usage

soil-fauna-ai is a CLI tool split into different subtool:

- Automatic segmentation
- COCO to Biigle converter
- Recursive file copy utility



## Segmentation

```
soilfauna segment
```

## COCO to Biigle converter

```
soilfauna coco2biigle
```

## File copy utility

```
soilfauna cpfiles
```
